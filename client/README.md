readme_content = """# TripVault - Travel Memory Journal

Welcome to **TripVault**! This project is my Week 1 submission for the Full Stack (MERN) Internship at CodGen. It’s a travel memory journal designed to help users securely log their trips, store photos, and keep their travel memories organized.

This week, I focused on setting up a robust backend, connecting to MongoDB Atlas, and building a secure authentication system from scratch.

---

## 📁 Project Structure
The repository is split into two clear sections:
- `server/` : The backend engine built with Node.js and Express.
- `client/` : The frontend interface built with React and Vite.

---

## 🛠️ Tech Stack
I’ve used the following technologies to build the foundation:
- **Backend**: Node.js, Express, MongoDB (via Mongoose), bcryptjs (for password security), and JWT (for authentication).
- **Frontend**: React (Vite), React Router DOM (for navigation), and Axios (for API communication).

---
